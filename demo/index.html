Install the Skill

Prerequisites

  • Claude Code CLI
  • Node.js 18+
  • ffmpeg
  • Kling AI API credentials

One-command install

The installer copies the skill and walks you through setting up your API keys:

git clone https://github.com/TamerinTECH/claude-skill-klingai-animation.git
cd claude-skill-klingai-animation

# Windows (PowerShell)
.\install.ps1

# Mac / Linux
chmod +x install.sh && ./install.sh

Use it

In Claude Code, type:

/animate-character ./character.png "typing excitedly, getting faster"

Claude will analyze your image, craft a Kling AI prompt, generate the video, and produce a sprite sheet PNG with ready-to-paste code.

Use the Output in Your Website

The skill outputs a single PNG sprite sheet + a JSON metadata file. To add the animation to any web page:

1

Copy the sprite sheet PNG into your project (e.g. /images/fox.png)

2

Add a <div> element where you want the animation

3

Paste the CSS snippet from the skill output

HTML

<div class="sprite-fox-coding"></div>

CSS

.sprite-fox-coding {
  width: 200px;
  height: 200px;
  background: url('fox-coding-sprite.png') 0 center no-repeat;
  background-size: 12200px 200px;
  animation: sprite-fox-coding-play 5.08s steps(61) infinite;
  overflow: hidden;
}

@keyframes sprite-fox-coding-play {
  from { background-position: 0 center; }
  to { background-position: -12200px center; }
}

That's it. No JavaScript, no libraries, no video files. Works in any browser, any framework, any static site.

React

<div
  style={{
    width: 200,
    height: 200,
    backgroundImage: "url('/images/fox-coding-sprite.png')",
    backgroundSize: '12200px 200px',
    animation: 'sprite-fox-coding-play 5.08s steps(61) infinite'
  }}
/>
Quality options: Use --quality=low for 8fps (lighter, ~40 frames) or --quality=high for 12fps (smoother, ~60 frames). The default is high.
